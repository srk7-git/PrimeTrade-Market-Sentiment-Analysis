Trader Performance vs Market Sentiment Analysis

Objective

Analyze the relationship between trader performance and Bitcoin market sentiment using Hyperliquid trading data and the Fear & Greed Index.

Methodology

- Data cleaning and preprocessing
- Date conversion and dataset merging
- Exploratory Data Analysis (EDA)
- Sentiment-based performance analysis

Key Findings

- Extreme Fear generated the highest average profit per trade.
- Extreme Greed showed the highest win rate.
- Fear periods had the largest average position sizes.
- Trading behavior varied significantly across sentiment regimes.

Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
